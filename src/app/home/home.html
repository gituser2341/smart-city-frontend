Core Features

Everything a Smart City Needs

Full-stack implementation built for transparency, speed, and real accountability.

📍

Geo-Location Mapping

Auto-captures precise location with OpenStreetMap embed — officers see exactly where to go.

🔴

Priority Escalation

LOW → MEDIUM → HIGH → EMERGENCY. Critical issues surface first, emergency cards pulse red.

📸

Image Proof Upload

Citizens attach photographic evidence. Officers see it on their dashboard with full context.

🤖

Auto-Assign by Department

Complaints auto-route to the right officer — WATER, ELECTRICITY, SANITATION, ROAD.

📊

Admin Analytics

Live bar charts, resolution rate donut, and department statistics for administrators.

🗺️

Map Preview on Cards

Every complaint card shows an embedded map pin — location visible instantly.

🔔

Notification System

Citizens get notified when complaint is assigned and when resolved by the officer.

🔐

JWT Role-Based Auth

Spring Security + JWT with CITIZEN, OFFICER, and ADMIN role-based access control.
